# 缺页归档

@{雾务局}的审问室没有窗，墙上却一直有水往下淌。

@{林照夜}坐在铁椅里，面前摆着他的身份证、退档通知、昨夜票夹和那张旧案车票。四样东西排得很整齐，像有人提前给他摆好了遗物。

对面的中年审查员翻了三遍记录，最后抬头问：“你确定自己叫林照夜？”

“不确定的话，我现在可以改名吗？”@{林照夜}说。

声音出口时有点哑。昨晚到现在他只睡了二十分钟，梦里全是站牌点阵在啃他的名字。他舌根发苦，像含着一枚没化开的旧硬币。

审问室角落有台旧式打字机，没人碰，纸页却自己往外吐。每吐一行，@{林照夜}桌上的身份证就淡一点。第一行是出生日期，第二行是住址，第三行开始只剩准考证号。

他伸手去按住身份证，指尖刚碰到塑封边，墙上的水痕忽然聚成一个小小的钟面。审查员抬起眼：“别碰。污染物会加速归档。”

“我还没死。”@{林照夜}说。

“档案里没有你活着的证据。”

审查员没有理会他的冷笑话，把身份证推回来：“系统里查不到你。学校档案也只剩准考证号。按规程，你可能是雾灾污染生成的人形残响。”

铁门外，@{乔声}的声音插进来：“他会疼，会怕，会欠便利店六块五豆浆钱。残响没有消费记录。”

她进门时左手手套湿了一半，像刚从水里捞过什么。审查员皱眉：“乔声，你没有资格担保。”

@{乔声}把一份临时报告放到桌上。“那就记我违规。反正旧案科最擅长把人记成没有。”

这句话让屋里安静了一瞬。

@{林照夜}抬头看她，第一次发现她眼底也有没睡醒的青色。审查员冷着脸去拿报告，桌上的旧案车票忽然自己翘起一角。

雾从车票背面冒出来，细得像线。它绕过四样证物，最后缠住@{林照夜}右耳。

第一下钟声落在铁桌里。第二下落在墙缝水痕里。到第十二下，审问室的灯全暗了，只剩车票背面的旧案编号发着红。

@{林照夜}没有躲。

他用拇指压住票角，疼得牙根发酸，却终于听清@{余仁}没说完的话：“旧库门不是终点……车上还有一个没下去的人……他拿走了名单。”

第十三下响起时，@{林照夜}看见一段不属于自己的记忆。

三年前雾灾夜，@{余仁}把十三路开进雾墙。车厢最后一排坐着一个戴灰帽的人。灰帽人没有脸，手里却拿着@{雾务局}的封条。

他把一册湿档案塞进投币箱，对@{余仁}说：“有人会听见钟，听见的人就是下一任活见证。”

@{余仁}没有接那册档案。他把车停在旧库门前，手指抖得几乎握不住方向盘。车厢里没有乘客说话，只有投币箱里传出一页页纸翻动的声音。

灰帽人弯下腰，把封条贴在投币箱口。“你开进去，这车就从线路上消失。你不开，名单上的人一个也活不过今晚。”

@{余仁}低头看了一眼驾驶台旁的家属照片。照片边缘已经被雾啃掉一角。他突然把乘车名单撕下一页，塞进刹车踏板底下，又用血在车票背后划了旧案编号。

“那就让后面的人听见。”他哑声说。

灯亮回来。

铁桌上多了一页湿透的乘车名单。名单第一行是@{余仁}，第二行被墨水糊住，只露出一个“乔”字。

@{乔声}的手猛地攥紧。

@{林照夜}把那页纸推到她面前，没有问她父亲的事。问出口就太轻了，压不住这张纸从三年前带回来的重量。

审查员终于变了脸色。他拿起名单，墙上的水痕同时退去一截，像审问室终于承认这页纸存在。

@{余仁}的失踪状态从系统里跳出来，后面不再是空号，而是四个字：`缺页待补`。

@{林照夜}靠回椅背，右耳彻底听不见了。左耳里却响起@{余仁}最后一口气：“别让名单进库。”

审查员的手悬在键盘上，半天没有落下。他终于把“污染残响”的临时判定删掉，换成另一行：`活见证人，需外勤监管`。

@{乔声}看见那行字，肩膀微不可察地松了一下。她没有道谢，只把裂开的黑线手套摘下来，露出掌心一道旧雾痕。雾痕形状像被人撕走半页纸。

@{林照夜}看了一眼那道痕，又看向名单上那个被墨水糊住的“乔”字。很多问题堵在喉咙里，最后只剩一句：“你父亲也坐过那班车？”

@{乔声}把手套重新戴上。“我来旧案科，就是为了查这个。”

半小时后，@{林照夜}拿到一张临时外勤协查证。

证件是纸质的，边角还没压膜，像@{雾务局}随时准备把它撕掉。@{乔声}站在走廊尽头等他，黑线手套上多了一道裂口。

她说：“余仁案只能算局部归档，名单上的第二个人还没确认。”

@{林照夜}把协查证塞进胸袋。纸面贴着湿衣服，有点冷，也有点真实。

走廊深处有钟声轻轻试了一下音。证件背面慢慢浮出下一行字：`活见证人 @{林照夜}，列入雾档第十四页。`
