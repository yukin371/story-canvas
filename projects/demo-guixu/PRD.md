# PRD: 归墟

## 产品定义

**书名**: 归墟
**类型**: 仙侠玄幻
**篇幅**: 中长篇，约12万字，25章，3卷
**布局**: layered（spec/ 目录分离）
**一句话概要**: 天枢宗首席弟子沈玄被师父夺取归墟体重生归来，必须在蝴蝶效应搅乱的记忆碎片中重新布局，一边复仇一边阻止归墟域封印崩塌。

## 商业蓝图

- **目标平台**: 起点中文网 / 番茄小说
- **连载模式**: 日更双章 → 周更精修
- **目标读者**: 18-35岁男性，仙侠网文核心受众
- **对标作品**: 《凡人修仙传》（理性流）、《求魔》（暗线流）
- **核心卖点**: 重生≠全知，蝴蝶效应让前世记忆不可靠，产生持续的悬念和不确定性
- **hook**: 重生第一刻就发现时间线已经偏移——前世不存在的陌生人拦住了他的路

## 质量门禁

- 每章 4000-6000 字
- 每章至少一个转折或信息增量
- 复仇线和封印线交替推进，不超过3章只走单线
- 角色行为必须符合已建立的性格，禁止突然降智
- 每卷末尾必须有一个改变格局的事件

## 验收标准

- [ ] 3卷结构完整，卷末有格局转折
- [ ] 主角复仇线收束，封印线有阶段性结论
- [ ] 核心角色（沈玄、顾长渊、叶清漪、宋怀瑾）有完整弧光
- [ ] 伏笔至少埋设5条，回收率≥80%
- [ ] 全文≥10万字
