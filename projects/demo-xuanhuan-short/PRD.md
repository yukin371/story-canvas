# PRD: 炉火照天门

> 状态: demo
> 说明: 用于固定这个玄幻短篇 demo 的商业定位与小闭环目标，不作为协议真相源。

## 1. 项目定义

- 书名: 炉火照天门
- primaryGenre: `fantasy`
- subGenre: `xuanhuan`
- styleTags: `web-serial`
- targetPlatform: `qidian`

## 2. 目标读者

- 喜欢升级流与宗门竞争的网文读者
- 喜欢“压制 -> 反压制 -> 再抬升”节奏的玄幻读者

## 3. 核心承诺

- 每章都有一次境界或处境推进
- 压制与反压制必须给出兑现
- 首卷小故事要形成一次宗门危机中的阶段性逆转

## 4. 商业钩子

- 废脉少年借炉火残篇重开修行路
- 将死之际点燃真火，直接拉出强钩子
- 三章完成一次压制到反压制的小闭环

## 5. 当前约束

- 不空转讲设定
- 世界规则与修行代价要服务冲突，不要脱离剧情独立说明
