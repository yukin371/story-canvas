# Gender-Bender-Yuri Overlay

适用定位：

1. `primaryGenre=romance`
2. `subGenre=gender-bender-yuri`
3. 常伴随 `identity-shift`、`new-body-adaptation`
4. 可兼容 `urban-school-life`、`trauma-rebuild`
5. 目标读者接近 `yuri-reader`、`identity-shift-reader`

证据基础：

1. `noval-data/analysis-workbench/synthesis/genre-overlays/gender-bender-yuri.md`
2. `noval-data/review-index.md`

## 当前高置信写法

1. 性转不应只停在外形反差，要持续改变主角被看待、被约束和被关系牵引的方式。
2. 开篇尽快立住主角最强的一条旧身份惯性，并让它在新身体和新规则里频繁失效。
3. 新身体要尽快接入学校、职场、监护或更强的现实关系网络，而不是长期只剩照镜子震惊。
4. 每章最好至少交付一次关系位移，而不是重复新身体笑点。
5. 若带创伤修复线，主角接手的不只是新身份，还包括旧伤口、旧关系和主动权回收。

## 节奏建议

1. 首章先写旧身份惯性撞上新身体限制，再给第一轮关系错位。
2. 前几章至少要完成一次“旧逻辑失效 -> 新关系重排 -> 主角被迫改线”。
3. 第一卷更稳的交付，是主角在新身体和新舞台中建立第一轮生存位置与关系锚点。
4. 若走创伤修复线，第一卷最好至少交付一次明确反击与一条核心关系修复起点。

## 常见误区

1. 性转只停在外形和笑点，没有持续改变关系模式。
2. 旧身份很强，却很快不再影响选择。
3. 新身体适应没有具体环境压力，只剩设定陈列。
4. 关系看似热闹，但没有真实位移和情绪黏性积累。
5. 系统任务只负责整活，不负责真正改变关系与退路。

## 与普适层的分工

这层补的是：

1. 旧身份与新身体的冲突驱动
2. 性转题材的关系重排与情绪黏性
3. 现实锚点与创伤修复的接入方式

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
