# Academy-Supernatural-Action Overlay

适用定位：

1. `primaryGenre=fantasy`
2. `subGenre=academy-supernatural-action`
3. 常伴随 `awakening`、`high-school`、`selection`
4. 可兼容 `gaokao-route`、`dual-talent`
5. 目标读者接近 `academy-action-reader`

证据基础：

1. `noval-data/analysis-workbench/synthesis/genre-overlays/academy-supernatural-action.md`
2. `noval-data/review-index.md`

## 当前高置信写法

1. 学院、觉醒、高考、秘境这些舞台要当成真实筛选器和资源分配器，而不是背景板。
2. 主角的系统或特殊天赋负责例外性，学院秩序负责兑现性。
3. 每次成长都最好跟一次公开比较、实战或资格变化绑定。
4. 若带双系天赋或多能力，不要让它只剩一个能力真在工作。
5. 世界危机要逐步改变主角选择，不能只在口头上存在。

## 节奏建议

1. 首章先写清觉醒分层、学校秩序和资源梯度。
2. 前几章至少要完成一次“能力提升 -> 公开兑现 -> 名次/资源/资格变化”。
3. 第一卷更稳的交付，是主角从普通学生跃迁成学院黑马，并拿到更高舞台门票。
4. 若走高考路线，倒计时和考试结果必须真正决定主角第一轮牌桌跃迁。

## 常见误区

1. 系统负责一切，学院秩序只是摆设。
2. 战斗很多，但赢了之后没有资格或资源变化。
3. 双天赋看起来很强，实则没有协同工作。
4. 校园竞争长期重复打榜，缺少向外扩张。
5. 公开考试或选拔只是过场，没有真正决定去向。

## 与普适层的分工

这层补的是：

1. 学院筛选与资源梯度
2. 例外天赋的公开兑现机制
3. 高考、秘境、选拔式舞台的节奏

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
