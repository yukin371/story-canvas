2. `subGenre=historical-court-politics`
3. 当前更贴近 `low-position-entry`、`prince-consort`、`hostage`、`factory-guard`、`debt-officialdom`

5. 范围护栏：
   - 适合低位入局、暗中起盘型朝堂权谋
   - 暂不覆盖纯内阁 / 纯党争 / 更高层帝王心术全貌

## 当前高置信写法

1. 开篇先立住主角“靠近权力中心却没有真正权力”的处境，不要只讲他很会权谋。
2. 受辱、藏锋、收人、经营和抬位要形成连续节奏。
3. 暗中积累不能只停在口头布局，必须逐步落到人才、钱粮、武力、情报或门客上。
4. 每次隐藏资源积累后，都要逐步改变公开牌桌里的位置。
5. 若带系统，它不能替代人心试探、制度摩擦和位置博弈。

## 节奏建议

1. 首章先写清当前身份、上级关系和最危险的一笔人情、债务或监视。
2. 前几章至少要完成一次“拿到底牌 -> 引发更高层关注或试探”。
3. 第一卷更稳的交付，是主角从可随手处理的棋子，升级为不能再被轻易处理的关键变量。
4. 低位身份要持续构成真实限制，不能只在开篇点题一次。

## 常见误区

1. 朝堂只是背景板，剧情实际滑成普通经营爽文。
2. 主角总在暗中布局，却没有公开牌桌位次变化。
3. 身份羞辱前期有用，后期不再形成限制。
4. 势力积累变成收菜列表，没有和政治危机建立因果。
5. 只剩风评梗和桃色丑闻，没有落到职位、差事和制度资源变化。

## 与普适层的分工

2. 隐藏资源如何转成公开牌桌位置

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
