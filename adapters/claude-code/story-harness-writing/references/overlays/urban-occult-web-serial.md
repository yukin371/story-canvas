# Urban-Occult Web-Serial Overlay

适用定位：

1. `primaryGenre=fantasy`
2. `subGenre=urban-occult`
3. `styleTags` 包含 `web-serial`
4. 常伴随 `folk-occult`、`career-fiction`
5. 目标读者接近 `qidian-reader`、`urban-fantasy-reader`、`folk-occult-reader`

证据基础：

1. `projects/demo-urban-occult-long/README.md`
2. `projects/demo-urban-occult-long/MODULE.md`
3. `projects/demo-urban-occult-long/project.yaml`

## 当前高置信写法

1. 开场优先给职业现场、异常事件和必须接活的压力，不要先空讲世界观。
2. 民俗要通过办事流程、禁忌、口口相传的规矩落地，而不是百科讲解。
3. 每章最好至少兑现一种推进：
   - 案件线索前进
   - 职业能力试用
   - 主线认知抬升
   - 代价或权限变化
4. 单元案更适合 `2到3` 章完成一次局部结算，同时继续抬主线。
5. 章节结尾优先留下：
   - 新异案入口
   - 旧案反咬
   - 父辈/旧城隍线索
   - 执牌资格、规矩或代价升级

## 节奏建议

1. 首章先让读者知道主角为什么得接这份活，以及这份活哪里危险、哪里专业。
2. 前几章不要只写“见鬼”，要写“怎么处理、谁负责、处理失败会怎样”。
3. 若带 `career-fiction`，职业质感要反复回场，不能只在开头点题一次。
4. 若带 `folk-occult`，优先写禁忌触发、仪式步骤、老行话压力，而不是堆神怪名词。

## Planning Starter

这部分和 `planning-primitives.md` 配合使用，回答“这个题材最稳的起手形状是什么”。

### 1. 角色位 starter

参考 `demo-urban-occult-long`，卷一级开局最稳的角色位通常包括：

1. 主角：有明确职业入口的夜班/基层从业者
2. 导师或上司：懂规则、懂旧账、能把主角拉进线的人
3. 勘验型搭档：负责把异案细节和证据往前拨
4. 单元关键证人：原本不在局里，却被异案点名的人
5. 前期执行层反派：熟规则、跑灰线，但不是真正顶层
6. 主线缺席核心：失踪前辈、旧案责任人、旧城隍系遗留人物

角色卡优先写：

1. `seed.archetype` 要带职业口
2. `seed.hook` 要带反差或麻烦
3. `profile.summary` 要写出他和这套夜间秩序的摩擦点
4. `arc.want / need / motivation` 要能推动卷一持续接案
5. `arc.milestones` 至少覆盖：被点名、被拉进局、拿到资格、主动补位或独立顶岗

### 2. 主角 starter

这一类主角最稳的不是“天生懂玄学”，而是：

1. 先有现实职业流程
2. 再被异常事件强行拉进局
3. 先想活得别太狼狈
4. 再被旧案、旧债或旧责任逼到不能旁观

若一上来就像老手天师，职业线和代价线都会变薄。

### 3. 三章开局骨架

适合卷一前 `3` 章起手：

1. 第 1 章：职业现场出异常，主角被正面点名，被迫接下第一趟活
2. 第 2 章：沿规则和旧账去追第一条线，确认主线碎片正在被人利用
3. 第 3 章：第一次正面截断流程或抢回关键残件，主角拿到临时资格，下一案抬出

对应的 chapter 任务通常是：

1. 先把“为什么是他”立住
2. 再把“这一行怎么处理”立住
3. 再把“他以后还得继续干”立住

### 4. 单元案骨架

`2到3` 章单元案通常按这个节奏最稳：

1. 入口章：异象封街 / 借道 / 点名，规则不稳，主角被拉进局
2. 追查章：沿香灰、账页、旧井、名册、税号之类具体线索追规则
3. 结算章：截断流程、收煞、救人或抢回残件，同时抬高主线认知

单元案结算后最好至少留下一个：

1. 更高层操盘者的影子
2. 父辈或旧案线索
3. 资格升级或责任升级
4. 下一单元案入口

### 5. Scene starter

这一类题材的 scene 最容易发力的不是“气氛描写”，而是：

1. 职业流程被异常打断
2. 按规矩办事时发现规矩被人改写
3. 用具体物件或票据把异案和旧账接上
4. 地上流程与地下规则同时推进

高频有效物件：

1. 棺材、冷柜、纸人、香灰
2. 账册、名录、票簿、税号
3. 旧铃、残印、巡夜录
4. 红轿、纸马、戏票、河灯

### 6. 不要怎么起手

1. 不要先讲一整章世界观史料
2. 不要主角刚进局就什么规矩都懂
3. 不要只有阴森气氛，没有职业流程
4. 不要把单元案写成纯支线，完全不碰主线旧账
5. 不要让关键证人和执行层反派都只剩功能，没有人的欲望和怕点

## 常见误区

1. 都市只是背景板，案件没有现代职业接口。
2. 民俗元素很多，但办案逻辑和代价系统很空。
3. 单元案长期不结算，只靠阴森气氛拖读者。
4. 主角持续旁观，被前辈或设定替代行动。

## 与普适层的分工

这层补的是：

1. 都市职业入口
2. 民俗禁忌的场景化表达
3. 单元案与主线并抬的连载节奏

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
