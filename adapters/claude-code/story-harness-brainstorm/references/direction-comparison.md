# Direction Comparison

当已经有 `2-4` 个可写方向，但还不知道该砍谁时，再读取本文件。

把比较当成取舍步骤，明确保留和淘汰。

## 核心标准

优先比较：

1. 追更动力是否清楚
2. 主角异常态是否强
3. 单章循环是否稳定
4. 第一卷交付是否可检查
5. 更大棋盘是否接得上

## Kill Rule

满足以下任一，就优先淘汰：

1. 只能靠设定新鲜感撑住，没有稳定单章循环
2. 第一卷交付说不清，只剩“成长”“变强”“揭开真相的一部分”
3. 主角异常态不够强，前三章很难立住
4. 需要大量解释世界观后才开始好看
5. 章尾续追问题会快速重复

## Choose Rule

优先选：

1. 三章内就能立住异常态和外部压力
2. 单章循环天然能反复成立
3. 第一卷能给到一个像样的小闭环
4. 主角解决局部问题时会自然引出更大棋盘

## 常见误判

优先避开这些误判：

1. 世界观看起来最宏大
2. 人设标签最多
3. 名词最酷
4. 一次性爆点最强，但后续循环最弱
