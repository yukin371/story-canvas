# Classic Genre Tags

这是项目创建前 brainstorming 用的“经典题材 / tag 起步索引”。

它的用途是：

1. 帮用户快速找到方向词汇
2. 帮 agent 先做题材分叉
3. 帮 `premise`、`hook`、`serial loop` 先有落点

## 使用规则

1. 一次优先给 `3-6` 个相关方向
2. 先按大类筛，再按 tag 细化
3. 题材词必须配合 `异常态 / 单章循环 / 第一卷交付` 一起说
4. 选完 tag 后，继续补主角异常态、单章循环和第一卷交付

## 先筛哪几条轴

如果用户说“想看经典 tag”，优先按这几条轴收窄：

1. 大类：玄幻、都市、西幻、轻小说、悬疑、恐怖、科幻、历史、女频、职业流
2. 主循环：升级、破案、接案、任务、关系推进、生存试错、项目竞争
3. 主压力：资源匮乏、身份压制、规则代价、关系错位、制度高压、公开竞争
4. 主交付：变强、破局、关系确认、通关、站稳脚跟、赢下阶段结果

如果这四条轴还没定，先把轴定下来，再往二级 tag 深挖。

## 大类起步

### 玄幻 / 仙侠 / 武侠

- 核心卖点：升级、资源争夺、规则突破、门派/江湖秩序变动
- 主角异常态：废材藏锋、异质传承、被逐出、背负禁忌身份
- 单章循环：遇事 -> 破局 -> 得失交换 -> 抬高下一层压力
- 第一卷交付：立住修炼路、打穿一个阶段门槛、接上更大势力棋盘
- 常见 tag：`xuanhuan` `xianxia` `wuxia` `academy` `sect` `revenge` `upgrade`
- 二级经典 tag：
  - `academy`: 学院、试炼、排位、师承竞争
  - `sect`: 门派、山门政治、资源分配、外门内门上升
  - `alchemy` / `crafting`: 炼丹、炼器、职业副线反哺主线
  - `beast-taming`: 灵兽伙伴、契约成长、资源与战斗双线
  - `revenge`: 羞辱/灭门/被逐后的反打
  - `invincible-start` / `hidden-power`: 先强后藏、身份伪装、信息差碾压

### 都市异能 / 都市志怪 / 职业抓诡

- 核心卖点：现代秩序下的异常事件、职业线、民俗/规则、主线阴谋
- 主角异常态：突然入行、半人半异、带禁忌工具、被迫接班
- 单章循环：接案/撞异 -> 追查 -> 解规则 -> 结算代价 -> 主线抬升
- 第一卷交付：立住职业身份与规则体系，给一个阶段性异案闭环
- 常见 tag：`urban` `urban-occult` `monster-of-the-week` `career-entry` `folk-horror`
- 二级经典 tag：
  - `career-entry`: 被迫入行、试用期、带教与行业规矩
  - `monster-of-the-week`: 单元怪谈、每章/每单元一案
  - `folk-horror`: 民俗禁忌、地方传说、祖辈遗留规则
  - `secret-bureau`: 官方组织、异事科、隐秘执法
  - `artifact-bond`: 绑定法器/遗物/牌位后被拖入局
  - `double-life`: 白天社会身份 + 夜间异常职业

### 西幻 / 剑与魔法 / 冒险队

- 核心卖点：队伍协作、世界探索、阶级/信仰/种族矛盾、成长冒险
- 主角异常态：低阶职业却背关键使命、半吊子天赋、被错误选中
- 单章循环：接任务 -> 遇阻力 -> 暴露世界一角 -> 队伍关系变化
- 第一卷交付：队伍成形、第一阶段旅程闭环、接上更大的世界危机
- 常见 tag：`western-fantasy` `party` `adventure` `magic-school` `kingdom-conflict`
- 二级经典 tag：
  - `party`: 队伍磨合、职业互补、分工和信任建立
  - `magic-school`: 学院训练、考试、社团/阶层竞争
  - `road-journey`: 公路冒险、地点切换带来世界展开
  - `kingdom-conflict`: 国家、领地、边境冲突升级
  - `dungeon-raid`: 迷宫、副本、遗迹探索
  - `chosen-one-twist`: 被选中但能力/身份不匹配

### 轻小说 / 校园 / 青春恋爱

- 核心卖点：角色反差、关系推进、轻喜剧节奏、日常里的情绪增量
- 主角异常态：关系错位、身份反差、被迫同居/同队、秘密暴露风险
- 单章循环：日常事件 -> 关系摩擦/靠近 -> 小爆点 -> 下一轮关系悬念
- 第一卷交付：主关系明显前进一步，关系位置发生可感变化
- 常见 tag：`light-novel` `school` `romcom` `youth` `tsundere` `ensemble-cast`
- 二级经典 tag：
  - `romcom`: 高密度误会、嘴硬心软、互动推进
  - `club-life`: 社团、活动筹备、共同项目促关系
  - `cohabitation`: 同居、同室、被迫绑定的高频互动场景
  - `secret-identity`: 马甲、隐藏能力、线上线下双身份
  - `ensemble-cast`: 多角色轮番出场、群像轻喜剧
  - `healing-youth`: 情绪疗愈、日常陪伴、低烈度成长

### 悬疑 / 推理 / 怪谈

- 核心卖点：谜面、信息差、真相翻转、规则压力、线索追踪
- 主角异常态：被卷入、被怀疑、掌握错误线索、与案件存在私怨
- 单章循环：出现异状 -> 收线索 -> 误判/反转 -> 留更大疑点
- 第一卷交付：解一个主谜团或阶段性真相，同时留下更大问题
- 常见 tag：`mystery` `detective` `suspense` `weird-tale` `closed-circle`
- 二级经典 tag：
  - `closed-circle`: 封闭场域、有限嫌疑人、连续排除
  - `weird-tale`: 怪谈规则、传言成真、真假难辨
  - `procedural`: 程序推进、证据链、调查流程
  - `amateur-detective`: 非职业卷入、靠特殊视角破局
  - `serial-case`: 多案串联成大谜团
  - `reversal-heavy`: 高反转、叙述陷阱、信息差翻面

### 惊悚 / 恐怖 / 无限流

- 核心卖点：生存压力、规则试错、身份危机、恐怖场景兑现
- 主角异常态：误入规则场、被系统/副本绑定、知道一点但不够活命
- 单章循环：触发规则 -> 试错 -> 伤亡/代价 -> 暂时存活 -> 更深规则暴露
- 第一卷交付：通关或逃出第一阶段，同时明确更大生存体系
- 常见 tag：`horror` `survival` `infinite-flow` `dungeon` `rule-based`
- 二级经典 tag：
  - `rule-based`: 明规则 + 隐规则 + 试错死亡代价
  - `infinite-flow`: 副本轮转、关卡晋升、队伍筛选
  - `body-horror`: 肉体异变、污染、不可逆代价
  - `psychological`: 怀疑、错觉、认知崩塌
  - `escape-room`: 密室、生路、限时逃脱
  - `sacrifice-loop`: 必须付代价才能通关

### 科幻 / 末世 / 赛博

- 核心卖点：技术代价、资源崩坏、制度压迫、生存组织化
- 主角异常态：掌握危险技术、免疫/缺陷体质、边缘身份、系统漏洞持有者
- 单章循环：资源/任务压力 -> 技术或制度对抗 -> 付代价 -> 生存位阶变化
- 第一卷交付：在新秩序里站住脚，或发现世界真实运转方式的一角
- 常见 tag：`sci-fi` `post-apocalypse` `cyberpunk` `survival-base` `mecha`
- 二级经典 tag：
  - `post-apocalypse`: 生存基地、物资、外部威胁与组织重建
  - `cyberpunk`: 企业控制、义体、数据与身份交易
  - `survival-base`: 建点、守点、招人、扩张
  - `mecha`: 机体、战术、后勤和驾驶成长
  - `time-loop-sci-fi`: 时间回环、技术试错、未来信息差
  - `hard-sf-pressure`: 技术约束直接驱动剧情

### 历史 / 权谋 / 朝堂

- 核心卖点：身份博弈、利益交换、时代变局、权力升降
- 主角异常态：寒门误入局、边缘人掌握关键筹码、旧案关联者
- 单章循环：局势变化 -> 试探/布局 -> 交换筹码 -> 新敌意浮现
- 第一卷交付：从局外人变成局中人，并赢下第一个阶段性政治结果
- 常见 tag：`historical` `court-politics` `military` `strategy` `revenge`
- 二级经典 tag：
  - `court-politics`: 朝堂攻防、站队、奏对与失势风险
  - `military`: 边军、战场、军功和后勤博弈
  - `strategy`: 智斗、布局、算计反算计
  - `revenge`: 冤案、家仇、旧主旧臣复起
  - `reform`: 制度变革、派系冲突、理想与现实碰撞
  - `merchant-power`: 商路、钱权勾连、地方到中央上升

### 女频成长 / 感情推进 / 宅斗宫斗

- 核心卖点：关系拉扯、身份处境、情绪兑现、成长与反击
- 主角异常态：婚约/家族困局、重来一次、身份被压制、情感目标错位
- 单章循环：关系事件 -> 情绪受压/回击 -> 身份位置变化 -> 下一轮拉扯
- 第一卷交付：主角拿到更稳定的位置，主关系或主要敌对线形成阶段结果
- 常见 tag：`female-growth` `romance` `rebirth` `household-intrigue` `palace-intrigue`
- 二级经典 tag：
  - `rebirth`: 重来一次、提前布局、改命与旧情旧怨
  - `marriage-first`: 先婚后爱、契约关系、利益婚姻
  - `slow-burn`: 低速升温、长期情绪积累
  - `household-intrigue`: 家宅资源、长辈压制、同辈明争暗斗
  - `palace-intrigue`: 后宫/深宅规则、位份、恩宠与生存
  - `healing-romance`: 安全感、互相修复、关系支持成长

### 娱乐圈 / 职业成长 / 项目流

- 核心卖点：项目推进、行业反馈、竞争排名、能力兑现
- 主角异常态：新人入局、黑历史缠身、天赋错位、资源稀缺
- 单章循环：接项目 -> 做方案/上场 -> 反馈与竞争 -> 名次/资源变化
- 第一卷交付：站稳行业位置或完成第一个代表性项目闭环
- 常见 tag：`showbiz` `career` `competition` `project-flow` `team-up`
- 二级经典 tag：
  - `competition`: 舞台、排名、淘汰、训练与反馈
  - `project-flow`: 接单、执行、复盘、下一单升级
  - `showbiz`: 热搜、路人盘、商务、黑粉与舆论
  - `startup`: 创业、团队、融资、生死节点
  - `esports`: 战队、训练、版本、赛事升级
  - `team-up`: 多人合作、内部摩擦、角色分工成长

## 二级 tag 常用组合轴

当用户已经定了大类，但还想继续细化时，优先从这些轴里各选 `1-2` 个：

1. 身份轴：`academy` `career-entry` `amateur-detective` `newcomer` `edge-outsider`
2. 结构轴：`monster-of-the-week` `serial-case` `project-flow` `road-journey` `dungeon-raid`
3. 压力轴：`rule-based` `revenge` `competition` `resource-scarcity` `secret-identity`
4. 关系轴：`party` `cohabitation` `marriage-first` `ensemble-cast` `team-up`
5. 气质轴：`folk-horror` `healing-romance` `reversal-heavy` `hard-sf-pressure` `court-politics`

选轴规则：

1. 每条轴优先选 `1-2` 个 tag
2. 先定“剧情怎么转”，再定“皮肤像什么”
3. 把表面名词接近的 tag 归成同类，只保留最有用的那个

## 按读者体验反推 tag

如果用户先说的是“我想要什么感觉”，先把体验翻成题材和 tag。

### 想要“持续爽、升级快、压迫感强”

- 优先大类：玄幻、仙侠、都市异能、末世
- 优先 tag：`upgrade` `academy` `sect` `career-entry` `survival-base`
- 追问重点：
  - 主角一开始被压在哪
  - 每章怎么完成一次小破局
  - 第一卷打穿哪道门槛

### 想要“怪、悬、一直想追”

- 优先大类：悬疑、怪谈、都市志怪、恐怖
- 优先 tag：`weird-tale` `closed-circle` `monster-of-the-week` `rule-based` `folk-horror`
- 追问重点：
  - 读者每章最想知道什么
  - 真相是逐层翻还是一次翻面
  - 第一卷是解一个谜，还是只是立大谜

### 想要“甜、拉扯、人物关系好嗑”

- 优先大类：轻小说、女频成长、校园恋爱
- 优先 tag：`romcom` `cohabitation` `marriage-first` `slow-burn` `ensemble-cast`
- 追问重点：
  - 主关系最卡在哪
  - 单章关系增量靠什么事件发生
  - 第一卷关系要推进到哪一步

### 想要“职业感、项目感、看主角做事”

- 优先大类：职业成长、娱乐圈、都市职业线、西幻冒险队
- 优先 tag：`project-flow` `competition` `career-entry` `team-up` `party`
- 追问重点：
  - 这份职业/项目每章怎么产生新任务
  - 外部评价系统是什么
  - 第一卷代表性项目是什么

### 想要“狠、算计、多方博弈”

- 优先大类：历史权谋、商战/项目流、部分都市异能或悬疑
- 优先 tag：`court-politics` `strategy` `revenge` `secret-bureau` `reversal-heavy`
- 追问重点：
  - 主角现在最缺哪张牌
  - 对手最强处在哪里
  - 第一卷赢下的是名分、资源还是阵营位置

## 组合提醒

这些组合需要更清楚地处理主循环和第一卷交付：

1. `closed-circle + road-journey`
原因：一个要封闭收束，一个要持续换地图，主循环容易互相打架

2. `slow-burn + competition-heavy`
原因：关系推进慢，外部节奏快，容易两头都不上不下

3. `rule-based + mystery-box + lore-heavy`
原因：规则、谜团、设定同时堆，开局极易解释过载

4. `invincible-start + survival`
原因：主角太强会吃掉生存压力，除非代价机制非常明确

5. `marriage-first + ensemble-cast + palace-intrigue`
原因：关系、群像、权斗都重，第一卷交付容易发散

## 默认 starter 套路

如果用户只想先拿几个“经典可写起点”，可优先给这些：

1. `urban-occult + career-entry + monster-of-the-week`
2. `xuanhuan + academy + revenge`
3. `light-novel + cohabitation + secret-identity`
4. `mystery + weird-tale + closed-circle`
5. `project-flow + competition + team-up`

这些 starter 的共同点是：

1. 开局异常态容易立
2. 单章循环比较清楚
3. 第一卷比较容易给阶段性交付

## 收束补齐

看到题材 tag 后，继续补：

1. 主角最不正常的状态
2. 读者为什么追
3. 单章怎么反复成立
4. 第一卷到底交付什么
