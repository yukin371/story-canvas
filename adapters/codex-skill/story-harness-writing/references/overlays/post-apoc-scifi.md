# Post-Apoc-Scifi Overlay

适用定位：

1. `primaryGenre=scifi`
2. `subGenre=post-apoc-scifi`
3. 当前更贴近 `wasteland-survival`、`district-order`、`team-building`
4. 目标读者接近 `post-apoc-reader`
5. 范围护栏：
   - 适合废土底层上升、特区秩序、现代末世团队生存
   - 暂不覆盖军事科幻、硬科幻或大规模军团推演全貌

证据基础：

1. `noval-data/archetypes/post-apoc-scifi/README.md`
2. `noval-data/analysis-workbench/synthesis/genre-overlays/post-apoc-scifi.md`

## 当前高置信写法

1. 开场优先给具体街区、交易、资源缺口和现实脏活，不要先讲宏大设定。
2. 主角的上升要先落到身份、住处、队伍、据点、资源或位置，不要空喊重建秩序。
3. 每次上升都要补一个更高层风险、债务、责任或势力暴露。
4. 世界观扩张后，也要继续保留生活压力，不要只剩抽象势力名词。
5. 团队、据点和资源盘要成为真实推进器，而不是背景挂件。

## 节奏建议

1. 首章先证明这个世界真的不好活，再给主角一个务实的第一阶段目标。
2. 前几章至少要完成一次“解决生存缺口 -> 换来更稳位置 -> 招来更高层风险”。
3. 第一卷更稳的交付，是主角从无资格的小人物，升级到站住脚的局内人，或拿到第一块可持续生存盘和第一批稳定同伴。
4. 废土和现代末世两条支线都要把代价、责任和秩序接驳写出来。

## 常见误区

1. 一开篇先讲设定，没有生活层的脏、穷和缺。
2. 主角上升只有收获，没有更大代价和暴露。
3. 世界扩张后，具体生存压力完全消失。
4. 势力冲突很多，却没有可感知现实后果。
5. 团队、据点和资源盘只做背景，没有承担章节推进。

## 与普适层的分工

这层补的是：

1. 废土生活压力的落地方式
2. 底层上升与秩序牌桌的接驳节奏
3. 团队、据点与资源盘的章节用途

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
