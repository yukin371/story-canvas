# Mystery-Supernatural-Horror Overlay

适用定位：

1. `primaryGenre=thriller`
2. `subGenre=mystery-supernatural-horror`
3. 常伴随 `folk-horror`、`ghost-breaking`
4. 可兼容 `apprentice-entry`
5. 目标读者接近 `folk-horror-reader`

证据基础：

1. `noval-data/analysis-workbench/synthesis/genre-overlays/mystery-supernatural-horror.md`
2. `noval-data/review-index.md`

## 当前高置信写法

1. 开篇先让异事找上门，不要把题材写成神佛图鉴。
2. 主角不能只是旁观怪事，要通过法门、民俗规则、师承遗物或设坛流程真正破局。
3. 每轮异事结束后，都应补一个新的禁忌、代价或更大秩序线索。
4. 主角成长要从半吊子或旁观者，逐步变成能独立处理基础异事的入门者。
5. 若带高人或师父，前三到五章内主角就要承担真实判断或执行价值。

## 节奏建议

1. 首章先写主角如何接触第一桩异事，以及为什么不能躲开。
2. 前几章至少要完成一次“判断真假 -> 查探规则 -> 设坛/法器破局”。
3. 第一卷更稳的交付，是解决一类鬼祟事件，并明确主角当前法门边界。
4. 单元异事之间要逐步拼出更大的奇人、道门或旁门版图。

## 常见误区

1. 只堆民俗名词和神佛设定，没有事件推进。
2. 鬼怪只负责吓人，不参与规则博弈和行动选择。
3. 主角成长太快，街头探异的烟火感迅速消失。
4. 每次破局都靠新能力硬压，调查与禁忌边界被削弱。
5. 高人戏份过重，主角长期只是跟班。

## 与普适层的分工

这层补的是：

1. 民俗异事的单元推进
2. 法门入行与规则破局
3. 单元异事如何接长线真相

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
