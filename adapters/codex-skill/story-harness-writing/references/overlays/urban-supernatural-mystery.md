# Urban-Supernatural-Mystery Overlay

适用定位：

1. `primaryGenre=fantasy`
2. `subGenre=urban-supernatural-mystery`
3. 常伴随 `identity-mystery`、`world-anomaly`
4. 与 `urban-occult-web-serial` 的区别：
   - 这份更偏身份谜团与都市异常双线
   - 不是职业民俗办事流
5. 目标读者接近 `urban-fantasy-reader`、`mystery-reader`

证据基础：

1. `noval-data/analysis-workbench/synthesis/genre-overlays/urban-supernatural-mystery.md`
2. `noval-data/review-index.md`

## 当前高置信写法

1. 开篇优先给强视觉、强意象和异常现场，不要先做数值型升级说明。
2. 身份谜团与世界规则谜团要双线推进，不能拆成两条松散支线。
3. 每章都应新增一层有效信息，而不是重复神秘气氛。
4. 标题或核心意象要持续回场，不能只在开场用一次。
5. 若带高考、落榜或公共体系失败者切口，要尽快给主角一次重新进入主流超凡舞台的机会。

## 节奏建议

1. 首章先让主角被异常正面点名，再给第一层身份或世界线索。
2. 前三章内至少要给一层明确答案，不能纯堆问题。
3. 第一卷更稳的交付，是揭开一层身份谜团，并让读者看懂一层世界异常规则。
4. 若走公共体系翻盘线，第一卷还应交付一次超凡资格或都市身份的真实变化。

## 常见误区

1. 氛围很强，但信息增量不足。
2. 标题意象只在开场出现，后续不再消费。
3. 身份谜团和世界谜团失去联动。
4. 前几章全是问题，没有阶段性答案回报。
5. 系统或公开舞台只做数值通道，没有转成身份变化。

## 与普适层的分工

这层补的是：

1. 都市异常的意象驱动
2. 身份谜团与世界规则双线推进
3. 阶段性答案的投放节奏

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
