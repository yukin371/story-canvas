# Female-Cultivation-Growth Overlay

适用定位：

1. `primaryGenre=fantasy`
2. `subGenre=female-cultivation-growth`
3. 常伴随 `female-lead`、`fate-rewrite`、`sect-conflict`
4. 可兼容 `bailan-strategy`、`strong-relationship`
5. 目标读者接近 `female-web-serial-reader`

证据基础：

1. `noval-data/archetypes/female-cultivation-growth/README.md`
2. `noval-data/analysis-workbench/synthesis/genre-overlays/female-cultivation-growth.md`

## 当前高置信写法

1. 开篇先让读者看见女主被什么旧关系、旧宗门或旧剧本困住，不要先空讲设定。
2. 改命不只写打脸，还要写资源、归属感和关系站位怎样逐步向女主倾斜。
3. 若卖点带 `摆烂`，要把它写成生存策略、关系策略或成长策略，而不是纯偷懒梗。
4. 若带系统、返利或机缘，它们只能放大女主的选择结果，不能替她省略翻脸、讨债和重新站队。
5. 若带宿敌、圣女或百合强关系线，关系推进要反过来影响修行路线和阵营选择。

## 节奏建议

1. 首章先立住“她为什么必须跳出原有结构”，再给第一次主动反制。
2. 前几章至少要出现一次：有人重新理解她、支持她、或被迫重新站队。
3. 第一卷更稳的交付不是单次赢回面子，而是完成第一轮身份、资源和新归属重构。
4. 轻松感可以减压，但不能把命运压力冲掉。

## 常见误区

1. 旧关系压迫不够痛，后续反打缺少价值。
2. 只有打脸结果，没有谁真正开始靠向女主。
3. 群像很多，但没有形成新的归属感主线。
4. 系统或机缘太顺，替代了女主处理关系和阵营代价。
5. 强关系线很亮，但长期和修仙成长分离。

## 与普适层的分工

这层补的是：

1. 女主改命节奏
2. 旧关系解构与新归属重建
3. 强关系线与修行站位的绑定方式

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
