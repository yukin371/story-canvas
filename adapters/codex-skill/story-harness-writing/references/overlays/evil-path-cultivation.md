# Evil-Path-Cultivation Overlay

适用定位：

1. `primaryGenre=fantasy`
2. `subGenre=evil-path-cultivation`
3. 常伴随 `dark-survival`、`taboo-power`
4. 可兼容 `nonhuman-judge`、`order-rebuild`
5. 目标读者接近 `dark-cultivation-reader`

证据基础：

1. `noval-data/analysis-workbench/synthesis/genre-overlays/evil-path-cultivation.md`
2. `noval-data/review-index.md`

## 当前高置信写法

1. 邪道力量要服务“代价型变强”，不是普通修仙换黑皮。
2. 开篇先写清主角被什么压迫到必须走邪路，黑暗秩序要真实压身。
3. 每次禁忌收益都要补身体、道德、关系或局势后果。
4. 若主角是秩序纠偏者或异常存在，也要写清权限边界、人间锚点和长期目标。
5. 每轮反杀后都要抬出更高层邪道秩序或更深污染，避免只剩重复虐杀。

## 节奏建议

1. 首章先写生存危机，再给第一次不得不吞下的禁忌力量。
2. 前几章至少要完成一次“禁忌求生 -> 代价兑现 -> 摆脱一层压迫”。
3. 第一卷更稳的交付，是主角从耗材、祭品或底层邪修，变成能主动反杀和脱离首层控制的人。
4. 若是非人视角变体，第一卷也应交付一次人间行动抓手或组织抓手。

## 常见误区

1. 只有血腥，没有真正的力量代价。
2. 黑暗世界很吵，主角却没有清楚执念或目标。
3. 主角太快习惯邪道，异化失去张力。
4. 邪修手段只负责猎奇，不能持续影响局势。
5. 异常高空视角长期没有人间锚点。

## 与普适层的分工

这层补的是：

1. 邪道力量的代价结构
2. 黑暗秩序压迫与摆脱首层控制
3. 非人或异常视角的人间落点

不要用它替代 `writing-universal.md` 或 `workflow-gates.md`。
